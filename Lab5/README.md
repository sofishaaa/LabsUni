# Lab5 – Тестування Wikipedia з Selenium WebDriver та Jest

## Налаштування
1. Ініціалізація проєкту:
  npm init -y
2. Встановлення залежностей: 
  npm install --save-dev jest selenium-webdriver
4.	У package.json додано скрипт: 
	"scripts": {
	  "test": "jest"
	}
Структура
Lab5/
  package.json
  package-lock.json
  README.md
  tests/
    wiki.test.js
5. Тести
Усі тести написані з використанням Jest та Selenium WebDriver (Chrome).
•	Завдання 2: відкриття головної сторінки Wikipedia, перевірка наявності пошукового поля та логотипу.
•	Завдання 3: введення тексту у пошукове поле, запуск пошуку, перевірка заголовку сторінки.
•	Завдання 4: використання різних локаторів (XPath, CSS, id) для пошуку елементів на сторінці статті Selenium.
•	Завдання 5: інтерактивні дії: 
o	Клік по внутрішньому посиланню та перевірка зміни заголовку.
o	Використання явних очікувань (until).
o	Отримання CSS властивостей (наприклад, колір заголовку).
6. Запуск
  npm test
Jest автоматично знайде всі файли з суфіксом .test.js у папці tests та виконає їх.
7. Результат
Усі тести проходять успішно:
•	4 тести, 1 тестовий набір.
•	Selenium відкриває Chrome, виконує дії та перевірки.
